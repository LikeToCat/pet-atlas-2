## 开发工具

### client

+ 语言：TypeScript
+ 框架：vue3 + vue-router + vuex + vuecli
+ 格式化：eslint + prettier
+ css预处理工具：scss
+ 工具库：axios



### server

+ 语言：JavaScript
+ 框架：Express
+ 数据库： MongoDB + Mongoose
+ 工具库：
  + cors
  + dotenv
  + express-mongo-sanitize
  + helmet
  + xss-clean
  + cross-env
